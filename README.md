## Overview

**zk-kya-verifier** is an open-source library for zero-knowledge proofs and privacy-preserving applications.

Zero-knowledge proof circuits for verifying agent identity without revealing private credentials.

Topics: `zkp` `identity` `kya`

## Features

- 🚀 Production-ready library targeting zk privacy use cases
- 🔧 Easy to integrate into existing projects
- 📦 Zero-configuration defaults with full customization support
- 🧪 Comprehensive test coverage
- 📖 Well-documented API with examples
